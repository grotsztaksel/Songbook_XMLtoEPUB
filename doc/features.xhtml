Features

  • Use set of XML files to create an EPUB compliant set of HTML + XML + metadata files
  • Automatically create a Table Of Contents (TOC)
  • Automatically create alphabetical & interactive index of songs
  • Automatically create alphabetical & interactive index of authors (music and lyrics)
  • Easy selection of different modes of chord annotation (above the lines or next to them - or none)
  • Automatically create links between songs of similar titles
  • Create links between songs selected by user
  • Insert regular HMTL pages (other than ones created for each song) in the songbook

Note

The SongEbook does not actually create the ready-to use e-book (see workflow) As mentioned above, it generates a set of HTML files plus metadata. The actual EPUB is a zip-compressed set of such HTML and metadata, and since there's lots of ZIP compressors out there, everyone can find his or her favourite to plug into the workflow to create the final file ready to use on the ebook reader
