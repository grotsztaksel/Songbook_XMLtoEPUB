Workflow

Steps required to put a songbook to the e-book reader

  1. Prepare/modify the input files in compliance with the expected schema
  2. Include additional files (images, htmls, etc) if necessary
  3. Run SongEbook providing the master XML file as input argument

The following steps must be executed using 3rd party tools. These, however are commonly available and there are lots of free programs that can execute the next steps

  1. ZIP the output directory and (and change the extension of the archive from .zip to .epub)
  2. [Optionally] Convert the epub to another ebook format
  3. Upload the result to the e-book reader
